# 小红书高质量文案创作技能

> 基于实际发布经验与社区反馈持续优化

## 核心理念

真实 > 营销感。小红书社区对AI生成内容和营销号语气极度敏感。

## 特性

- ✅ 事实核查清单：防止CVPR/SIGGRAPH类混淆错误
- ✅ 语气控制：去除AI味，像真人说话
- ✅ 领域适配：3D/AI/半导体/消费电子等专业领域模板
- ✅ 发布前自检：3个核心问题快速排雷
- ✅ 避坑指南：基于真实评论区反馈总结的常见错误

## 使用场景

- 科技产品发布会文案
- 技术分析内容
- 行业趋势解读
- 产品测评文案
